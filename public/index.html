How It Works

Three steps to publish-ready content

No editing skills required. The engine handles everything from crop ratios to captions.

1
Drop Your Media
Upload raw safari footage, photos, audio or search for a GIF. Any format, up to 500 MB. The engine probes the file and detects dimensions, duration, and quality automatically.
Video · Photo · Audio · GIF
2
Engine Processes
FFmpeg crops to every platform's exact aspect ratio, boosts safari colour grade, applies the Lavira watermark, and generates 6 platform variants in parallel. Sharp handles images. Claude writes the caption.
FFmpeg · Sharp · Claude AI
3
Download & Post
Review your branded variants, edit the AI caption if needed, hit Approve, and download per-platform files. Or connect social tokens and publish directly. History is logged automatically.
Instagram · TikTok · Facebook · Twitter
⚡
Auto Mode (Daily)
The scheduler runs every day at 06:00 EAT and generates a fresh promo package automatically — cycling through all 8 safari destinations so your feed never repeats. Zero user input needed.
Scheduler · Smart Rotation · SQLite Memory
🤖
MCP Agent Mode
Connect Claude Desktop to the 13-tool MCP server and let an AI agent run the entire pipeline autonomously. Just say "Process today's Masai Mara video" and it handles everything.
Claude Desktop · JSON-RPC · 13 Tools
